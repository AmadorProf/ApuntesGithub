# Apuntes de Git y GitHub

Guía práctica para aprender Git y GitHub desde configuración básica hasta flujos de trabajo colaborativos. Pensado para alumnado de FP de grado superior.

## Índice

| # | Módulo | Qué aprendes |

| 1 | [Introducción](1-introduccion.md) | Qué es Git, control de versiones distribuido, casos de uso |

| 3 | [Comandos Básicos](3-comandos-basicos.md) | `init`, `clone`, `add`, `commit`, `log`, mensajes de commit |
| 4 | [Ramas y Fusiones](4-ramas.md) | Ramas, `merge`, `rebase`, resolución de conflictos |

| 7 | [Ejemplos Prácticos](7-ejemplos.md) | Repo desde cero, contribuir a open-source, equipo |
| 8 | [Stash, Tags y Releases](8-stash-tags-releases.md) | `git stash`, etiquetas, publicar releases |

## Cómo usar estos apuntes

Lee en orden. Cada módulo asume el anterior. Si ya tienes base, salta al módulo 4 (ramas) o al 6 (flujos avanzados). Ejecuta los comandos tú mismo: leerlos no sustituye a practicarlos.

## Recursos externos

- [*Pro Git* (libro gratuito)](https://git-scm.com/book/en/v2) — guía completa y en profundidad.
- [Documentación oficial de Git](https://git-scm.com/docs/git) — referencia de todos los comandos.
